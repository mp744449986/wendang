## 项目概述

在线文档浏览系统，采用 SSG 架构，支持 PDF/PPT/Word 文档上传、自动拆图、静态页面生成、广告位管理等功能。

**技术栈**：
- 后端：Node.js + Express + PostgreSQL
- 前端管理后台：Vue 3 + Vite + Pinia + Element Plus
- 部署：Docker Compose + Nginx
- 公开站点：纯静态 HTML（SSG 生成）
